સુચન :

  • માત્ર .xlsx ફાઈલ જ Upload કરવી.
  • ફાઈલ નું નામ ‘District.xlsx’ રાખવું.
    જેમકે Ahmedabad.xlsx
    અથવા
    અમદાવાદ.xlsx
  • ફાઈલમાં Worksheet ના નામ નીચે પ્રમાણે આપવા.
    Rural માટે : ‘District-Rural’ જેમકે ‘Ahmedabad-Rural’ અથવા ‘અમદાવાદ-Rural’
    Urban માટે : ‘District-Urban’ જેમકે ‘Ahmedabad-Urban’ અથવા ‘અમદાવાદ-Urban’
    Nagarpalika માટે : ‘District-Nagarpalika’ જેમકે ‘Ahmedabad-Nagarpalika’ અથવા ‘અમદાવાદ-Nagarpalika’
  • શરૂઆતમાં કે પછી કોઈ પણ બે column ની વચ્ચે ખાલી column રાખવી નહિ.
  • પ્રથમ row માં column ના નામ જ હોવા જોઈએ. અન્ય કોઈ પણ પ્રકારના headings આપવા નહિ.
  • Village ના નામ ની સાથે કોઉંસ i.e. ‘()’ માં કોઈ પણ પ્રકારની comment રાખવી નહિ.
  • EMamta Code ની સામે Village નું નામ હોવું જ જોઈએ.
  • Spreadsheet માં location નો data ‘tree structure’ પ્રમાણે જ હોવો જોઈએ.
  • Area નું નામ Village ના નામ પછી ની row થી શરુ થવું જોઈએ.
  • જો એક કરતા વધારે Area ના નામ એક જ cell માં હોય તો તેમની વચ્ચે ‘+’ નો ઉપયોગ કરવો. Space i.e. ‘ ‘ અથવા comma i.e. ‘,’ નો ઉપયોગ કરવો નહિ.
  • ASHA નું નામ હમેશા કોઉંસ i.e. ‘()’ માં જ લખવું. ‘{}’ અથવા ‘[]’ નો ઉપયોગ કરવો નહિ.
  • ASHA ના નામ માટે કોઉંસ i.e. ‘(‘ શરુ કર્યા પછી તેને પૂરો કરવો i.e. ‘)’ જરૂરી છે. ASHA ના નામ પછી Area ના નામ કે અન્ય કોઈ પણ પ્રકારની વિગત લખવી નહિ.
  • Area નું નામ ‘-‘ કે અન્ય કોઈ પણ પ્રકારના સિમ્બોલ થી શરુ થતું ન હોવું જોઈએ.
  • ASHA નું નામ નક્કી ના હોય તો ‘ખાલી જગ્યા’ હમેશા કોઉંસ i.e. ‘()’ માં જ લખવું.
  • Error ફાઈલમાં જરૂરી બદલાવ કરી તેને ફરીથી Upload કરી શકાય છે.
  • અન્ય કોઈ પણ પ્રકારની મુશ્કેલી / સહાયતા માટે (૦૭૯) ૨૩૨૮૭૦૨૮ Extn:૩૧૬ નંબર પર ફોન કરવો.
